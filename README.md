CodeAlpha Cyber Security Internship
Task 1: Network Sniffer 
Project Description
This project is a Python-based Network Sniffer developed as part of the CodeAlpha Cyber Security Internship. The tool captures network packets and displays source and destination IP addresses in real time.
Features
Captures network traffic
Displays source IP addresses
Displays destination IP addresses
Real-time packet monitoring
Technologies Used
Python
Scapy
How to Run
Install Python
Install Scapy using: pip install scapy
Run the program: python NetworkSniffer.py
Output
The program captures network packets and displays source and destination IP addresses in real time.
